# InspoBooks 📚

An Android app for tracking and discovering inspiration.

## Features
- 🔐 User authentication via Firebase
- 📖 Save and view personal inspiration boards
- 🔍 Search for inspiration images using an image API
- 🌗 Dark/light mode support

## Tech Stack
- **Language:** Kotlin
- **IDE:** Android Studio
- **Authentication:** Firebase Authentication
- **Database:** Firebase Cloud Firestore
- **UI:** XML Layouts
- **Testing:** Android Emulator

## Getting Started
1. Clone the repository
2. Open in Android Studio
3. Connect your own Firebase project via `google-services.json`
4. Run on an emulator or physical device
